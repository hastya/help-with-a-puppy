# 🐶 Help with a puppy

Веб-приложение для персонализированного ухода, обучения и медицинского
сопровождения собак на основе **породы** и **возраста** питомца.

Реализовано как full-stack веб-приложение (адаптация мобильного ТЗ под
web-интерфейс): REST API на Node.js + Express, база SQLite, и одностраничный
адаптивный интерфейс в стиле **Material Design 3** с поддержкой тёмной темы.

## ✨ Возможности

| Раздел | Что реализовано |
|--------|-----------------|
| **Аккаунт** | Регистрация по email/паролю, вход (JWT), онбординг-опросник (опыт, цель) |
| **Питомцы** | Неограниченное число собак, карточка (кличка, порода, дата рождения, пол, стерилизация, вес, аватар), **автоопределение статуса** (щенок / взрослая / пожилая), переключатель профилей (табы, как в Netflix) |
| **Здоровье** | Трекер веса с графиком и **сравнением с нормой породы**, автоматический **календарь прививок**, аптечка (лекарства и обработки), особенности и болезни породы |
| **Дрессировка** | План обучения, **зависящий от возраста** (щенок / взрослый), чек-лист команд со статусами (не начато → в процессе → освоено), учёт минут тренировок |
| **Питание** | **Калькулятор калорий** (формула RER = 70 × вес⁰·⁷⁵ + коэффициенты возраста/стерилизации), расчёт разовой порции в граммах, график кормлений, персональный **стоп-лист** опасных продуктов |
| **Ветеринар** | Интерактивный **справочник симптомов** с оценкой срочности, кнопка **SOS** с круглосуточными клиниками, ветеринарный паспорт (документы) |
| **Отчётность** | Дашборд по каждой собаке (динамика веса, освоение команд, расходы, тепловая карта активности), **сводная аналитика** по всем собакам, **AI-инсайты**, экспорт **CSV** и печать/PDF |
| **Настройки** | Профиль хозяина, единицы (метр./имперская), тема (светлая/тёмная/системная), язык, гранулярные **уведомления** и тихие часы, учёт расходов, архивация/удаление профиля, экспорт данных и **удаление аккаунта (GDPR)** |

## 🏗️ Технологический стек

- **Backend:** Node.js + [Express](https://expressjs.com/)
- **База данных:** [SQLite](https://www.sqlite.org/) через [better-sqlite3](https://github.com/WiseLibs/better-sqlite3) (без внешней СУБД — нулевая настройка)
- **Аутентификация:** JWT ([jsonwebtoken](https://github.com/auth0/node-jsonwebtoken)) + [bcryptjs](https://github.com/dcodeIO/bcrypt.js)
- **Frontend:** ванильный JavaScript SPA (без сборки), CSS в стиле Material 3
- **Графики:** [Chart.js](https://www.chartjs.org/) (подключается локально, без CDN)

> Оригинальное ТЗ описывает Android-приложение (Kotlin/Flutter + NestJS/FastAPI +
> PostgreSQL). Здесь та же предметная логика перенесена в самодостаточное
> веб-приложение, которое разворачивается одной командой. См.
> [«Соответствие ТЗ»](#-соответствие-тз).

## 🚀 Быстрый старт (локально)

Требуется **Node.js ≥ 18**.

```bash
git clone https://github.com/hastya/help-with-a-puppy.git
cd help-with-a-puppy
npm install
npm start
```

Откройте <http://localhost:3000>, зарегистрируйтесь и добавьте первую собаку.

> Подробная инструкция по развёртыванию (Docker, systemd, Nginx, облако) —
> в файле **[DEPLOYMENT.md](DEPLOYMENT.md)**.

## 📱 Android-приложение (автономное, офлайн)

Кроме веб-версии, есть **мобильное приложение для Android**, которое можно
скачать на телефон и запустить — оно работает **полностью офлайн**, данные
хранятся на устройстве, сервер не нужен.

- **Скачать APK:** раздел **[Releases](../../releases)** → `help-with-a-puppy.apk`
  (собирается автоматически через GitHub Actions).
- **Как установить и как это устроено:** см. **[MOBILE.md](MOBILE.md)**.

Исходники мобильной сборки — в каталоге [`mobile/`](mobile/) (Capacitor + PWA).

## 🔌 REST API (кратко)

Все эндпоинты, кроме `/api/auth/*`, требуют заголовок `Authorization: Bearer <token>`.

| Метод | Путь | Назначение |
|-------|------|-----------|
| POST | `/api/auth/register` · `/login` | Регистрация / вход |
| GET | `/api/breeds` | Справочник пород |
| GET/POST | `/api/pets` | Список / создание собак |
| PUT/DELETE | `/api/pets/:id` | Редактирование / удаление |
| GET/POST | `/api/pets/:id/health/weights` | Трекер веса |
| GET/POST | `/api/pets/:id/health/vaccinations` | Прививки |
| GET/POST | `/api/pets/:id/health/medications` | Лекарства и обработки |
| GET/PUT | `/api/pets/:id/training` | План дрессировки |
| GET | `/api/pets/:id/nutrition/calories` | Калькулятор рациона |
| GET | `/api/pets/:id/nutrition/stoplist` | Стоп-лист продуктов |
| GET/POST | `/api/vet/symptoms` · `/symptoms/assess` | Справочник симптомов |
| GET | `/api/reports/pets/:id/dashboard` | Дашборд по собаке |
| GET | `/api/reports/summary` | Сводная аналитика |
| GET | `/api/settings/export` | Экспорт всех данных (GDPR) |

```bash

```

Приложение проверено сквозным браузерным сценарием (регистрация → добавление
питомца → здоровье/дрессировка/питание/ветеринар → отчётность с графиками →

## 📋 Соответствие ТЗ

Реализованы ключевые функциональные блоки ТЗ «Help with a puppy»: управление
несколькими питомцами с индивидуальными породой/возрастом, персонализированные
разделы (здоровье, дрессировка, питание, ветеринар), раздел «Отчётность» с
интерактивными графиками и AI-инсайтами, и раздел «Настройки» с гранулярными
уведомлениями и GDPR-удалением. Мобильные интеграции (FCM push, Google Fit,
Bluetooth-весы, платежи Google Play) вынесены за рамки веб-версии.
